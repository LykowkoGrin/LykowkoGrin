Какой из меня, к чёрту, программист, если я ненавижу программировать? Если для меня это — мука болезненная, постыдное занятие, что-то вроде выдавливания геморроя. Ведь я раньше думал, что от моих программ что-то становится лучше. Да не нужны они никому! Я сдохну, а через два дня меня забудут и начнут жрать кого-нибудь другого. 
Андрей Тарковский(edited byme)
